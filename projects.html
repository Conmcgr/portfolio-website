NBA Award Prediction

This project is a sophisticated web application that leverages machine learning to predict NBA Most Valuable Player (MVP) and Rookie of the Year (ROY) awards. It demonstrates proficiency in data science, web development, and sports analytics.

Key Features:

  • Data Collection: Scrapes player statistics from Basketball Reference, covering seasons from 1977 to 2024.
  • Data Preprocessing: Cleans and normalizes player statistics, including per-48-minute performance metrics.
  • Machine Learning: Utilizes Random Forest Regressors to predict MVP and ROY vote shares, with model calibration using Isotonic Regression.
  • Web Interface: Flask-based frontend allowing users to upload CSV files with player statistics and receive award predictions.
  • Interactive Results: Displays top 5 predictions with an option to toggle between simplified and full statistical views.

Technical Stack:

  • Python for data processing and machine learning
  • Flask for web server
  • HTML/CSS/JavaScript for frontend
  • Pandas and NumPy for data manipulation
  • Scikit-learn for machine learning models
  • Beautiful Soup for web scraping
